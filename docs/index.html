🎯 Visão Geral do Projeto

Este projeto serve como referência para estudantes de Programação Orientada a Objetos e Qualidade de Software. Simula um ambiente de batalha naval onde os jogadores estrategicamente posicionam navios e afundam a frota inimiga.


O jogo é jogado num grelha 10×10. O sistema de coordenadas é definido como:

(x, y) ∈ {0, …, 9} × {0, …, 9}

🚀 Instalação e Configuração

Pré-requisitos

JDK 21 ou superior · Git · Docker (opcional)

  1. Clonar o repositório
    git clone https://github.com/IGE-112244/Battleship2.git
  2. Entrar na pasta
    cd Battleship2
  3. Compilar e executar
    mvn package -DskipTests
java -jar target/BattleshipGamePlayer-2.0.jar
  4. Ou via Docker
    docker run -it carol123456/battleship-game:latest
